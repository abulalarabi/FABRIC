#### Hand and Hinges
* Scale everything to 80%
* Print hinges only with TPU -> 5~10% infill
* Print everything else with PLA
* 14 Hinges are required for one hand.
* No support required for palm.
* Use support material for fingers

#### Wrist and Pulleys
* Print everything with PLA.
* Use 0.1mm or 0.15mm layer height for the pulleys.
* Requires 5 pulleys for each hand.
